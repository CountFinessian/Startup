Implementation

WINDOWS

Julia is only available for Windows 7 or later for both 32 and 64 bit versions

  1. Type Download Julia into your search engine.
  2. Click either 32-bit or 64-bit download for windows, depending on your computer size.
    *Note: 64-bit download will only work with 64 bit computers.
  3. Click the new download (shown below).
  4. Hit Next
  5. Make sure the “Add Julia to PATH” checkbox is checked then hit Next
  6. Check finish box


MAC

For macOS 10.9 Mavericks or later releases.

  1. Type Download Julia into your search engine.
  2. Click 64-bit download for macOS
  3. Find julia-1.8.5-mac64.dmg file in folder
  4. Drag file to Applications Folder


SETUP/TEST

Note: Julia can be run either in your terminal or on the Julia app. The layout is the exact same.

TO OPEN TERMINAL

  1. Search “terminal” in search bar/applications
  2. Type “julia” and hit ENTER. This screen should pop up
  3. Type println(“Hello World”)
    The command line should output .

    4. Hello World

Congrats, you have successfully installed the Julia Program! For additional documentation and syntax help, visit me.
Fill out Instruction Evaluation when finished!

TO UNINSTALL

  1. Open applications folder
  2. Go to Julia App Settings
  3. Click Uninstall
